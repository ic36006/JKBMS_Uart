📖 คู่มือการใช้งาน: กล่องส่งข้อมูล JK BMS เข้า Google Sheets
อุปกรณ์ตัวนี้ทำหน้าที่เป็น "ตัวกลาง" ที่คอยดึงข้อมูลสุขภาพแบตเตอรี่จาก JK BMS แล้วส่งขึ้นไปเก็บบน Google Sheets ผ่านอินเทอร์เน็ตแบบอัตโนมัติ เพื่อให้คุณสามารถดูกราฟหรือตรวจสอบสถานะแบตเตอรี่จากที่ไหนก็ได้
📊 1. อุปกรณ์ส่งค่าอะไรขึ้นไปบ้าง?
กล่องนี้จะอ่านค่าสำคัญๆ จากแบตเตอรี่ส่งขึ้นไปให้คุณ ดังนี้ครับ:
•	โวลต์รวม (Total Voltage) และ กระแสไฟ (Current)
•	กำลังไฟ (Power) ที่กำลังใช้งาน (วัตต์)
•	เปอร์เซ็นต์แบตเตอรี่ (SOC %)
•	อุณหภูมิ (Temperature): ทั้งเซนเซอร์จุดที่ 1, จุดที่ 2 และอุณหภูมิของตัวบอร์ด BMS (MOS Temp)
•	ความจุที่เหลืออยู่ (Remaining Capacity) และ จำนวนรอบการชาร์จ (Cycle Count)
•	ค่าของเซลล์ย่อย (Cells): โวลต์เฉลี่ย, ความห่างโวลต์ (Diff) และโวลต์ของแบตเตอรี่แต่ละเซลล์แยกกัน (ตั้งแต่เซลล์ที่ 1 ถึง 16)
⏱️ 2. ความถี่ในการส่งข้อมูล
•	เมื่อตั้งค่าเสร็จแล้ว อุปกรณ์จะทำการส่งข้อมูลขึ้น Google Sheets ทุกๆ 30 วินาที อย่างต่อเนื่องตลอดเวลา
•	มีระบบดูแลตัวเอง: หากเน็ตบ้านหลุด หรือไฟตกจนเราเตอร์ดับเกิน 5 นาที อุปกรณ์จะทำการ "รีสตาร์ทตัวเอง" อัตโนมัติ เพื่อพยายามเชื่อมต่อเน็ตใหม่ให้กลับมาทำงานได้โดยที่คุณไม่ต้องเดินไปถอดปลั๊กเสียบใหม่
⚙️ 3. วิธีตั้งค่าใช้งานครั้งแรก (Setup)
เมื่อได้อุปกรณ์ไปครั้งแรก หรือต้องการเชื่อมต่อกับ WiFi ใหม่ ให้ทำตามขั้นตอนนี้ครับ:
	1.	จ่ายไฟให้อุปกรณ์ (เสียบปลั๊ก)
	2.	หยิบโทรศัพท์มือถือขึ้นมา เข้าไปที่หน้าค้นหาสัญญาณ WiFi
	3.	หา WiFi ที่ชื่อว่า JK_BMS_Setup แล้วกดเชื่อมต่อ
	4.	โทรศัพท์จะเด้งหน้าต่างตั้งค่าขึ้นมาอัตโนมัติ (หากไม่เด้ง ให้เปิดแอปเว็บบราวเซอร์ เช่น Safari หรือ Chrome แล้วพิมพ์ช่องค้นหาว่า 192.168.4.1)
	5.	กดปุ่ม Configure WiFi 6. เลือกชื่อ WiFi บ้าน/ที่ทำงานของคุณ และ ใส่รหัสผ่าน WiFi ให้ถูกต้อง
	6.	ในช่อง Google App Script URL ให้เอาลิงก์ของ Google Sheets ที่คุณสร้างเตรียมไว้มาวางใส่ลงไป
	7.	กดปุ่ม Save อุปกรณ์จะบันทึกข้อมูลและรีบูตตัวเอง 1 รอบ จากนั้นมันจะเริ่มส่งข้อมูลทันที!
🔄 4. วิธีล้างค่าเครื่อง (Factory Reset)
หากคุณต้องการเปลี่ยนไปใช้ WiFi บ้านอื่น, เปลี่ยนรหัสผ่าน WiFi ใหม่ หรือเปลี่ยนลิงก์ Google Sheets คุณต้องทำการล้างค่าความจำในเครื่องก่อน โดยทำดังนี้:
	1.	ในขณะที่เครื่องเปิดทำงานอยู่ ให้ กดปุ่มที่ตัวเครื่องค้างไว้ประมาณ 3 วินาที (ปุ่มที่กำหนดไว้ หรือปุ่ม BOOT บนตัวบอร์ด)
	2.	ปล่อยมือ เครื่องจะทำการล้างข้อมูลการเชื่อมต่อเดิมทิ้งทั้งหมดและรีสตาร์ทตัวเอง
	3.	อุปกรณ์จะกลับไปปล่อย WiFi ที่ชื่อ JK_BMS_Setup ออกมาอีกครั้ง เพื่อให้คุณเริ่มตั้งค่าใหม่ตั้งแต่ขั้นตอนที่ 1 ได้เลย
